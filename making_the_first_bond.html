Making the first bond

Setup

Introduction

Once we mounted the motor drivers on the frame of the machine (see the motor control section) and got the master-slave communication working (see the system architecture section) we were ready to perform the first bond test.

The MCU we used was a NUCLEO-H723ZG board. The whole setup was powered using three bench power supplies as we haven't yet designed a suitable custom power supply for the whole apparatus. You can see a photo of the setup in the image below.

STM32CubeMonitor control interface

As we didn't have the master PCB ready yet nor the LCD screen and the camera, we used the STM32CubeMonitor software to create a basic control UI as described in the images below.
The bond is perfomed by pressing the "bond" button, which turns on the ultrasonic driver to full power for 250ms. This timing has to be precise: if it lasts too little the wire doesn't have enough time to melt; if it lasts too long the bond gets formed and then detaches due excessive stimulation.

Performing the bond

The bonding prcedure was perfomed by following the following steps:
  • Position the bed by manipulating the XY coordinates;
  • Make the head descend by manipulating the Z coordinate;
  • Close the clamp solenoid;
  • Activate the ultrasonic driver;
  • Open the clamp solenoid;
As the Z motor was not yet calibrated, the descent of the machine head was aided by manual human intervetion. The XY movement, on the other hand, required no outside help.
